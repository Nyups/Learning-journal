Header section

We are in the Header element, used to define the ... header of a document section.

A Header element code look like this :

            
                <header>
                <h1>Header section</h1>
                </header>

------------------------------------------------------------------------------------------------------------------

Main section

We are in the Main element, used to contain the main content of a Web page

a Main element code look like this :

            
                <main>
                <h1>Main section</h1>
                </main>

------------------------------------------------------------------------------------------------------------------

Section element

We now are in a Section element, used to divide content into smaller section.

A Section element code look like this :

                
                    <section>
                    <h1>Some title</h1>
                    </section>

We can also assign ID or Class to it to style it or link it like so :

                
                    <section id="part-01" class="section">
                    <h1>Some title</h1>
                    </section>
