Header section

We are in the Header element, used to define the ... header of a document section.

A Header element code look like this :

            
                <header>
                    <h1>Header section</h1>
                </header>

------------------------------------------------------------------------------------------------------------------

Main section

We are in the Main element, used to contain the main content of a Web page

a Main element code look like this :

            
                <main>
                    <h1>Main section</h1>
                </main>

------------------------------------------------------------------------------------------------------------------

Section element

We now are in a Section element, used to divide content into smaller section.

A Section element code look like this :

                
                    <section>
                        <h1>Some title</h1>
                    </section>

We can also assign ID or Class to it to style it or link it like so :

                
                    <section id="part-01" class="section">
                        <h1>Some title</h1>
                    </section>

------------------------------------------------------------------------------------------------------------------

Nav element

This is a nav element. A section with navigation link.

A nav element code would look like this

                
                    <nav>
                        <ul>
                            <li><a>nav 1</a></li>
                            <li><a>nav 2</a></li>
                        </ul>
                    </nav>

And display like this :

  • Nav 1
  • Nav 2

    Text Elements

    Bellow are some of the text element / modification used in HTML :


  • The em element

    The em element mark the text as stress (italic) emphasis.
    an em element code would look like this :

                    
<p>I am quite certain that this was <em>your</em> doing.</p>

    Which will display like this :
    "I am quite certain that it was your doing."

  • Idiomatic Text Element

    the Idiomatic element is used to highlight alternative voice or mood,
    idiomatic,technical term or thoughts.

    An Idiomatic code example would look like this :

                        
<p>Like the japanese always say : <i lang="jp"> がんばる (頑張る) </i> and do your best!</p>

    which would display as :
    "Like the japanese always say : がんばる (頑張る) and do your best!"

    The "lang" attribute inside is used to specify the language of the content. in our case , Japanese.

  • Strong and B element

    -The P element marks text as strong importance.
    -The B element is used to bring attention for text that have low importance fot the meaning of the content.

    An example of Strong and B element would be :

                        
// Strong example:
<p>
    <strong>Warning:</strong> I have peanuts allergy !
</p>

// B example
<p>
    Don't you hate how <b>Chips</b> taste like nothing?
</p>

    which would display as :
    Warning: I have peanuts allergy !
    Don't you hate how Chips taste like nothing?

    ... it's the same isn't it?
