Nav and navbar in HTML

The <nav></nav> element is used to provide a navigation links to other section in the document or other sections in the website.
A lot of times we will see the <nav> element used for menus or toble of contents.

An example of using the <nav> would look like this :
<nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#about">About</a></li>
    <li><a href="#contact">Contact</a></li>
  </ul>
</nav>
